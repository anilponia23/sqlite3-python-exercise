
# Manufacturing Data Platform (Python + SQLite)

## Overview
This project implements a small data platform for a fictional manufacturing company using **Python** and **SQLite**.  
It supports:
- A normalized schema for products, machines, work orders, and operations.
- Demo data for realistic queries.
- SQL queries to answer common business questions.
- Python functions for data access and reporting.
